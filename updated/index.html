Privacy Policy for Easy Custom Autofill

Last updated: April 29, 2026

Easy Custom Autofill is designed for saving low-risk text snippets that you type often, then filling them into text fields from the browser right-click menu.

This extension is not intended for passwords, payment information, medical information, private identifiers, recovery codes, banking details, or other sensitive information.

Data Collection and Storage

Saved Autofill Entries

When you create an autofill entry, the entry name and saved text are stored locally in your browser using chrome.storage.local.

Your saved autofill entries are not sent to me, to an analytics service, or to an external server by this extension.

Local Extension Preferences

The extension may store small local preferences, such as whether you have clicked the Privacy Policy link or dismissed a local notice. These preferences stay in your browser and are not sent to an external server.

No Analytics or Tracking

Easy Custom Autofill does not use analytics, advertising trackers, tracking pixels, remote logging, or user behavior tracking.

No Remote Styling Services

The extension does not load Water.css, Google Fonts, jsDelivr, remote CSS libraries, remote fonts, or styling CDNs. The popup and FAQ styles are bundled with the extension.

How the Extension Uses Your Data

The extension uses your saved entries only to provide its main feature:

  • displaying your saved entries in the popup
  • creating right-click menu items
  • copying a saved entry when you click Copy
  • inserting your selected saved text into a field when you choose it from the right-click menu
  • exporting and importing your saved list when you choose those options

The extension does not sell, rent, share, transmit, or disclose your saved entries to third parties.

Important Website Notice

When you insert saved text into a website field, that website may be able to read, process, save, submit, or share that text just like anything you manually type or paste into the website.

For example, if you autofill text into a form, chat box, search box, login field, comment box, or other webpage input, the website and any scripts running on that website may be able to access that field depending on how the website works.

Easy Custom Autofill does not control what websites do with text after you place it on their page.

For this reason, this extension should only be used for public, low-risk, non-sensitive text.

Permissions

Storage

Used to save your autofill entries locally in your browser.

Context Menus

Used to add Easy Custom Autofill options to the right-click menu on editable fields.

Active Tab and Scripting

Used to insert your selected saved text into the field you are using.

Content Script Access

The extension includes a content script so it can detect editable fields and fill text on webpages. This is needed for the autofill feature to work on normal inputs, textareas, search boxes, and editable text areas.

This access is not used to collect browsing history, monitor your activity, or send page content to an external server.

Export and Import

Export

When you use Export List, the extension creates a JSON backup file containing your saved entries.

This exported file is plain text. Anyone with access to that file may be able to read its contents.

Keep exported backups somewhere safe and do not export sensitive information.

Import

When you use Import List, the extension reads the selected JSON file locally in your browser and imports valid entries.

Importing may replace your current saved list. You should export your current list before importing another backup.

Sensitive Data Warning

Do not save sensitive information in Easy Custom Autofill.

Do not save:

  • passwords
  • one-time codes
  • recovery phrases
  • payment card information
  • bank information
  • Social Security numbers or government ID numbers
  • medical information
  • private personal details
  • anything you would not want stored in a plain-text backup file

Easy Custom Autofill is not a password manager and should not be used as one.

Security Limitations

Easy Custom Autofill is built to keep your saved entries local, but local storage does not mean risk-free.

Your saved entries or exported backups could potentially be accessed because of things outside this extension's control, including:

  • physical access to your device
  • malware on your device
  • malicious browser extensions
  • browser or operating system vulnerabilities
  • compromised websites where you paste or autofill text
  • unsafe backup storage
  • accidental sharing of exported JSON files

You are responsible for keeping your device, browser, extensions, and backup files secure.

Data Loss

Saved entries may be lost if you uninstall the extension, clear browser or extension data, reset your browser profile, switch devices without importing a backup, or experience browser storage corruption.

Regularly export your saved list if the entries are important to you.

Third-Party Services

The extension itself does not use third-party analytics, advertising, tracking services, remote fonts, remote CSS libraries, or CDNs.
